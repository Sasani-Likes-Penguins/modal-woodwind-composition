### ~~Minor Issue 2~~: Kaba—Moldavian vs. Bulgarian distinction—**Status: RESOLVED**
The guide previously referenced *kaba* (Bulgarian open-rimblown technique) as if applicable to the user's Moldavian fipple kaval. The user confirmed these are distinct instruments and techniques. All references to *kaba* in the guide have been corrected to describe the Moldavian dual-voice technique accurately. **Status: RESOLVED ✅**
